# ユーザーアプリ エントリポイント

このディレクトリには、機能ごとに分類された実行可能なアプリケーションエントリポイントが含まれています。

## ディレクトリ構成

- `basic/`
  - ランタイムライフサイクルの基本
  - シグナルハンドリングの基本
  - PER + PHM の基本
- `communication/`
  - `someip/`
  - `zerocopy/`
  - `dds/`
- `feature/`
  - `runtime/`: ライフサイクル重視のフィーチャーアプリ
  - `can/`: SocketCAN 受信/デコード フィーチャーアプリ
  - `ecu/`: ECU フル統合 フィーチャーアプリ

## 注記

- 再利用可能なサポートモジュールは `user_apps/src/features/` にあります。
- すべてのアプリエントリファイルはオンボーディングと再利用のために完全にコメント付きです。
- ホスト側の DoIP/DIAG テスターは `tools/host_tools/doip_diag_tester/` に分離されています。
